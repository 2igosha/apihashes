## Api Hashes plugins

### IDA plugin

You need the IDA SDK (7.3+) to compile it. Use the Makefiles and LIB files provided with the SDK to compile apihashes\_ida.cpp.

### Ghidra script

Copy ApiHashesScript.java to %GHIDRA\_DIR%/Ghidra/Features/Base/ghidra\_scripts. Then locate it via the Script Manager by name and run it.
